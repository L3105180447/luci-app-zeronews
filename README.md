luci-app-zeronews
一个为 OpenWrt 设计的 ZeroNews (零讯) 客户端 LuCI 配置界面。

本项目专门针对 OpenWrt 21.02 及更早版本 进行优化，采用传统的 Lua + CBI 架构，确保在旧版固件或资源受限的设备上拥有良好的兼容性和响应速度。

简介
ZeroNews 是一款简单、高性能的内网穿透工具。通过本插件，你可以轻松地在 OpenWrt 路由器上管理 ZeroNews 隧道，实现远程访问内网服务、远程控制及数据传输。

主要功能
全版本兼容：完美适配 OpenWrt 21.02, 19.07 等传统 LuCI 界面。

配置直观：支持 Token 设置、日志查看、跳转官网。

状态监控：实时显示 ZeroNews 运行状态及日志输出。
